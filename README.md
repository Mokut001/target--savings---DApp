# Cardano Target Savings DApp

A DApp that helps users save ADA towards a specific goal within a timeframe.

## Features
- **Goal Calculator**: Calculates monthly savings needed.
- **Smart Vault**: Locks funds in a Plutus V2 script.
- **Progress Tracking**: Real-time progress bar and "amount remaining" display.
- **Secure Withdrawal**: Only accessible once the target is met.

## Deployment (Termux/Vercel)
1. `cd frontend`
2. `npm install`
3. Connect your Wallet (Nami/Eternl).
4. Deploy `frontend/` to Vercel via GitHub.
